## Install

Using npm:

```sh
npm install @runtime-edge/ponyfill --save
```

or using yarn:

```sh
yarn add @runtime-edge/ponyfill --dev
```

or using pnpm:

```sh
pnpm install @runtime-edge/ponyfill --save
```
